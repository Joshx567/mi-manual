Elementos-Semanticos Descripcion Documentacion
<html> Representa la raiz del elemento Documentacion
<header> Describe contenido destinado a la parte superior o encabezado de la pagina Documentacion
<meta> Meta nos dice la informacion del documento HTML por medio de atributos Documentacion
<link> Link nos especifica las relaciones entre el documento actual y un recurso ajeno a la pagina. Documentacion
<body> Body representa el contenido del documento HTML. Documentacion
<div> Div es un contenedor generico para elementos de categoria flow, se le puede modificar su estilo con css Documentacion
<article> Encierra contenido independiente del resto de la pagina web Documentacion
<section> Es usado para dividir el contenido dentro de un elemento article Documentacion
<nav> Ayuda a proporcionar enlaces de navegacion Documentacion

<footer> Es un pie de pagina para su ancestro sectioning mas cercano(article, aside, nav, section) o para body si no hay ancestro Documentacion
<main> Main es donde se muestra el contenido principal de la pagina, no deberia haber mas de un main Documentacion
<span> El elemento es un container generico que no representa inherentemente nada, usualmente usado para agrupar elementos con fines de estilo Documentacion
<audio> El elemento audio como su nombre indica, es usado para introducir audio Documentacion
<img> El elemento img tambien como su nombre indica, es usado para introducir imagen Documentacion
<video> El elemento video como se indica, muestra un video en la pagina web Documentacion

<sub> Sub se usa para permitir subindices como log2e por ejemplo Documentacion
<strong> Strong es una forma de negrilla para los textos Documentacion
<em> Em es cursiva, esto es usado para dar enfasis a lo que se habla Documentacion
<mark> El elemento mark permite resaltar texto de interes Documentacion
<q> El elemento q indica que el texto encerrado es una cita corta en línea Documentacion
<blockquote> Similar al q, solo que es para una cita extensa Documentacion
<cite> Cite permite incluir el nombre de un trabajo (Un libro, un ensayo, un poema, una película, una pintura, un programa de computadora, etc.) Documentacion
<dfn> Dfn se utiliza para identificar el término a definir, mientras que la definición del término se coloca fuera del elemento. Documentacion
<abbr> Abbr sirve para para identificar una abreviatura Documentacion
<small> Small permite definir la letra chica y los comentarios secundarios de una pagina Documentacion
<s> S permite tachar letra, util para decir cosas que se han actualizado y que el anterior dato esta mal Documentacion
<code> Code sirve para mostrar codigo con su propio formato Documentacion
<samp> Samp se utiliza para encerrar texto que representa la salida de un programa de computadora Documentacion
<kbd> Kbd se utiliza para representar una imagen parecida a la entrada de texto de un teclado Documentacion
<pre> Pre se utiliza para contiene texto que debe presentarse exactamente como está escrito en el archivo HTML Documentacion

Selectores Descripcion Documentacion
Universal(*{ }) Se cambian los datos para todos los valores del html Documentacion
P Seleccionamos para todos los parrafos <p> Documentacion
.foo Segun las clases que tengas, las puedes llamar en el espacio foo y asi modificar tu clase creada Documentacion
#foo Selecciona un elemento pero con el id Documentacion
a[foo] Permite seleccionar elementos en función de la presencia o del valor de un determinado atributo Documentacion
selector:pseudo-class { … } Son pseudo-clases que permite seleccionar elementos en base a su estado (:disabled), posicion de los elementos en el documento(:nth-child) e interacción del usuario(:hover) Documentacion
selector::pseudo-elemento { … } Seleccionan parte de un elemento en lugar del elemento mismo. Los pseudo-elementos se añaden a los selectores y permiten añadir estilos a una parte concreta de los elementos seleccionados. Por ejemplo, el pseudo elemento ::first-line selecciona solo la primera línea del elemento especificado por el selector Documentacion
Union de selectores A B {} Seleccionamos todos los elementos seleccionados por B que son descendientes de los elementos seleccionados por A Documentacion
Union de selectores A>B{} Seleccionamos todos los elementos seleccionados por B que son hijos de los elementos seleccionados por A Documentacion
Union de selectores A~B{} Seleccionamos todos los elementos seleccionados por B que son hermanos de los elementos seleccionados por A Documentacion
Union de selectores A+B{} Seleccionamos todos los elementos seleccionados por B que son hermanos y que están inmediatamente despues de los elementos seleccionados por A. Selecciona el elemento B solo si sigue inmediatamente a A. Los elementos A y B son hermanos Documentacion
Especificidad Podemos usar diferentes combinaciones de selectores para seleccionar un mismo elemento, sin embargo la esfecificidad se encaerga de decidir que valor sera aplicado, para saber mas sobre su comportamiento, la documentacion Documentacion
Longitudes Descripcion Documentacion
px Pixel, unidad de medida en css Documentacion
rem Relativo al tamaño de la letra del elemento root (El elemento html) Documentacion
% Relativo a otro valor. Si establecemos el valor de la propiedad font-size, será un porcentaje del font-size del elemento padre, si usamos un porcentaje para establecer el valor del elemento width, será un porcentaje del width del elemento padre Documentacion
vw Relativo al ancho del viewport Documentacion
vh Relativo al alto del viewport Documentacion
